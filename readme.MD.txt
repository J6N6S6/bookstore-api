# bookstore-api

API for a bookstore application in C#

## Funcionalidades

- Cadastro de livros
- Consultas de livros
- Autenticação JWT com dois níveis de acesso (admin e público)

## Tecnologias Utilizadas

- **ASP.NET Core**
- **Entity Framework Core**
- **JWT** (JSON Web Tokens)
- **PostgreSQL** (ou SQL Server, dependendo do seu banco de dados)

## Instalação

### Requisitos

- **.NET SDK** (versão X.X)
- **PostgreSQL** ou **SQL Server**
- **Node.js** (se usar frontend com NPM)
